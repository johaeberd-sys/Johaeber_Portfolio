## 📋 Required Images

### 1. Hero Image (Your Profile Photo)
- **Filename:** `hero.png`
- **Recommended Size:** 400x400px (square)
- **Format:** PNG (with transparent background) or JPG
- **Purpose:** Displays in the hero section
- **Tips:**
  - Use a professional photo
  - Clear face, good lighting
  - Solid or transparent background
  - High quality, not blurry

### 2. Project Images
- **Filenames:** 
  - `project1.png`
  - `project2.png`
  - `project3.png`
- **Recommended Size:** 800x400px (2:1 ratio)
- **Format:** PNG or JPG
- **Purpose:** Project showcase screenshots
- **Tips:**
  - Use clean screenshots
  - Show your best work
  - Crop to relevant content
  - High resolution

## 🎨 Image Guidelines

### File Sizes
- **Target:** Under 200KB per image
- **Maximum:** 500KB per image
- **Why:** Faster loading, better performance

### Optimization Tools
- [TinyPNG](https://tinypng.com) - Free compression
- [Squoosh](https://squoosh.app) - Advanced options
- [ImageOptim](https://imageoptim.com) - Mac app
- [RIOT](https://riot-optimizer.com) - Windows app

### Formats
- **PNG:** Best for graphics, logos, transparency
- **JPG:** Best for photos, smaller file size
- **WebP:** Modern format, best compression (optional)

## 🖼️ Creating Project Screenshots

### For Web Projects
1. Open your project in browser
2. Use full-screen mode
3. Take screenshot (Windows: Win+Shift+S, Mac: Cmd+Shift+4)
4. Crop to show main content
5. Resize to 800x400px
6. Optimize file size

### For Design Projects
1. Export from design tool (Figma, Adobe XD, etc.)
2. Use high-quality export settings
3. Resize to 800x400px
4. Optimize file size

### For Other Projects
1. Create a visual representation
2. Use mockup generators if needed
3. Ensure clear, professional appearance
4. Resize and optimize

## 🔧 Troubleshooting

### Image Not Showing?
1. Check filename matches exactly (case-sensitive)
2. Verify file is in `public/images/` folder
3. Check file extension (.png, .jpg, not .PNG or .JPG)
4. Clear browser cache (Ctrl+Shift+R)

### Image Too Large?
1. Use compression tool (TinyPNG)
2. Resize to recommended dimensions
3. Convert to JPG if currently PNG
4. Reduce quality slightly (80-90%)

### Image Looks Blurry?
1. Use higher resolution source
2. Don't upscale small images
3. Export at 2x size, then resize down
4. Use PNG for sharp graphics
