## 🎨 Image Guidelines

### Recommended Specifications
- **Size**: 800x600px or 1200x900px (4:3 ratio)
- **Format**: PNG or JPG
- **File Size**: Under 500KB per image
- **Quality**: High resolution, clear and professional

### What to Include
- Behind-the-scenes production photos
- Equipment setup shots
- Live streaming setups
- Studio/workspace photos
- Event coverage photos
- Team collaboration shots
- Technical equipment close-ups
- Before/after production shots

## 🔧 How It Works

1. **Click "View Images" button** on any work experience card
2. **Modal opens** showing images in a gallery
3. **Navigate** using arrow buttons or keyboard arrows
4. **Close** by clicking X, overlay, or pressing Escape

## 📝 Customizing Images

### To Add More Images:
Edit `js/data.js` and add more image paths to the `images` array:

```javascript
{
    company: "Your Company",
    position: "Your Position",
    duration: "2020 - Present",
    responsibilities: [...],
    images: [
        "public/images/experience1-1.png",
        "public/images/experience1-2.png",
        "public/images/experience1-3.png",
        "public/images/experience1-4.png",
        "public/images/experience1-5.png",
        "public/images/experience1-6.png",
        "public/images/experience1-7.png",
        "public/images/experience1-8.png",  // Add more if needed
        "public/images/experience1-9.png"   // Add more if needed
    ]
}
```

### To Remove Images:
Simply remove the image paths from the array or remove the entire `images` property to hide the "View Images" button.

## 🖼️ Image Optimization

### Before Adding Images:

1. **Resize** to recommended dimensions
2. **Compress** using tools like:
   - [TinyPNG](https://tinypng.com)
   - [Squoosh](https://squoosh.app)
   - [ImageOptim](https://imageoptim.com)
3. **Rename** to match the naming convention
4. **Test** in the modal viewer

## 💡 Pro Tips

1. **Consistency**: Use similar aspect ratios for all images
2. **Quality**: Use high-resolution images that look good on large screens
3. **Relevance**: Choose images that showcase your actual work
4. **Privacy**: Blur or remove any sensitive information
5. **Branding**: Include your logo or watermark if desired
6. **Storytelling**: Choose images that tell the story of your work
7. **Variety**: Mix wide shots, close-ups, and action shots
